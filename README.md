# Daniyal Pharmacy Dashboard

Modern React + Material UI pharmacy management frontend for Daniyal Pharmacy.

## Run

```bash
npm install
npm run dev
```

## Demo Login

- Admin: `admin@pharmacy.com` / `Password123!`
- Super Admin: `superadmin@pharmacy.com` / `Password123!`

## Features

- Responsive dashboard, CRUD pages, POS, reports, profile and settings
- Light/dark mode with persistence
- Sidebar collapse, sticky navbar, snackbar notifications
- MUI DataGrid tables, Recharts charts, dialogs and reusable UI blocks
